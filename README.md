# NHA RAG Assistant
A local AI assistant built with Ollama, LangChain, and Streamlit.

## Team
- Moahmed Ebrahim
- youssef Hesham
- moahmed Hassan
- Lina Essam

## Features
- - Generate & Explain Code
- Written Code Highlight 
- Support More Files (DOCX, PPTX, XLSX, Markdown)
- Image Analysis & OCR (Vision)
- Streaming Responses
- Clear & Export Chat
- Voice Assistant (Like ChatGpt)
- Web Search

## Setup
pip install -r requirements.txt
ollama pull qwen2.5-coder:1.5b
streamlit run app.py
